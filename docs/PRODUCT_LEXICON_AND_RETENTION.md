# Лексика и удержание внимания

**Статус:** канон продукта (2026-07-20)

## Запрещённый мета-язык (никогда в UI)

Нельзя показывать пользователю: ИИ, AI, LLM, нейросеть, «генерация», «сгенерировано»,
«модель», «промпт», «стиль текста», «тип текста», «алгоритм», «хаб», «Immersive Hub».

Вместо этого — язык дня и жизни: «собрать», «открыть», «как прожить», «чего ждать».

## Психологический контракт экрана «Сегодня»

Причина открыть приложение каждый день:

1. **Ориентация** — чего ждать / чего не ждать.
2. **Действие** — один выполнимый шаг.
3. **Осторожность** — где не давить.
4. **Сферы** — что сегодня сильнее / слабее (ротация по дню, без шаблона).
5. **Признание** — текст про этого человека и его окружение, не «для всех».

## Визуальный эталон

Профиль (`ProfileV2SystemScreen`) — единый тон: тёплый фон, display-заголовки,
секции с одной задачей, без карточного шума и без «продуктового жаргона».

## Таро

Название поверхности: **Таро** (ритуал ясности). Не «хаб».
