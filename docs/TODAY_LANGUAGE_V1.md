## 1. Главная ось качества

| Ось | Роль |
|-----|------|
| **Банальность / небанальность** | **Главный фильтр.** Текст либо рисует сцену, либо отклоняется. |
| Эзотерика / рациональность | **Не фильтр.** Мотивация, Вселенная, предназначение, перемены — **разрешены** и ожидаемы для аудитории. |
| Абстракция / конкретика | Производная от банальности: абстракт без жизненного контекста = слабый текст. |

**Диагноз типичного AI-текста:**

- звучит как генератор цитат;
- взаимозаменяем между людьми и днями;
- безопасный, но **не вызывает эмоцию**;
- человек читает и через 10 секунд забывает.

**Целевое ощущение:** пользователь **узнаёт себя** или **видит сцену** — как в хорошем коучинге, астрологии, таро, self-development.

## 2. Три правила (стек качества)

Правила проверяются **вместе**. RULE_001 — необходимое, но **не достаточное** условие.

### TODAY_WRITING_RULE_001 — правило кино

> **Каждая значимая фраза должна создавать узнаваемую жизненную ситуацию, образ или внутренний диалог.**

**Проверка:**

> *Могу ли я представить конкретную сцену из жизни человека?*

| Ответ | Вердикт |
|-------|---------|
| **Да** — ситуация, жест, мысль, разговор, привычка | ✅ проходит RULE_001 |
| **Нет** — только красивая мысль | ❌ fail RULE_001 |

**Контрпример:** «Сегодня вы можете открыть холодильник и понять, что вам пора меняться» — сцена **есть**, но текст **плохой**. → см. RULE_002, RULE_003 и `cal-004` в calibration dataset.

### TODAY_WRITING_RULE_002 — узнаваемость

> **Пользователь должен иметь возможность сказать: «Да, такое действительно бывает».**

Не абсурдная сцена ради образа. Не выдуманный микро-мoment. Узнаваемый паттерн из реальной жизни аудитории Today (отношения, работа, внутренний спор, ожидание, усталость).

### TODAY_WRITING_RULE_003 — эмоциональный крючок

> **Фраза вызывает хотя бы одну эмоцию** — иначе читается как служебное сообщение.

| Эмоция | Примеры эффекта |
|--------|-----------------|
| **надежда** | «может появиться возможность» |
| **любопытство** | «неожиданно напомнит о себе» |
| **облегчение** | «проще заметить, что вы устали…» |
| **предвкушение** | «ответ, который давно откладывали» |
| **узнавание** | «снова возвращаетесь к одному разговору» |
| **вдохновение** | «освободить место для следующего шага» |

**Значимые фразы** — headline, subline, core message, insight, guidance, вечер с контекстом.  
**Исключения:** короткие императивы-действия, факты неба (S0), системные подписи UI.

### TODAY_WRITING_RULE_004 — точка зацепки *(CALIBRATION_HYPOTHESIS, не gate)*

> **Фраза должна цепляться хотя бы за одно:** сцена · ставка · узнавание · конкретная задача · конкретный объект · конкретное решение.

| | Пример | Зацепки |
|---|--------|---------|
| ❌ | Сегодня важно сохранять ясность. | нет |
| ✅ (слабее) | Выбор одного приоритета поможет избежать перегрузки. | проблема · действие |
| ✅ | Подготовь отчёт, который уже давно откладывал. | объект · действие · узнавание |
| ✅ (сильнее) | Разговор, который откладывали из-за страха услышать ответ. | сцена · ставка |

**Статус:** `CALIBRATION_HYPOTHESIS` — operationalized as `anchor_types[]` (`AT-001`…`AT-006`) in calibration schema v0.5. Каталог: [TODAY_ANCHOR_TYPES_V0.md](./TODAY_ANCHOR_TYPES_V0.md).

**Эволюция гипотезы (batch_3, 2026-06-23):** «хорошая фраза **обязательно** содержит ставку» **не выдержала** api_live keep/exemplar (фокус, перегруз, дедлайн, объекты без `stake_type`). Ставка остаётся **сильным усилителем**, не единственным правилом.

**Legacy sub-hypothesis — stake:** `stake_type` в dataset v0.3; batch_1/2 показали rewrite при scene без stake; batch_3 показал keep **без** stake при object/action/recognition.

## 2.1 Категории сцен (таксономия для калибровки)

Сцены группируются для **TL-0 dataset**, не как жёсткий gate на старте:

| Категория | Подтипы |
|-----------|---------|
| **Коммуникация** | сообщение · звонок · разговор · конфликт · признание |

| **Отношения** | ожидания · обиды · возвращение человека · поддержка |
| **Внутренний мир** | сомнения · усталость · мотивация · выбор |
| **Случайности** | совпадения · знаки · неожиданные встречи |

После 100+ размеченных примеров — уточнить таксономию по тому, **какие сцены чаще в score 7–10**.

## 4. Запрещено

| # | Паттерн | Почему |
|---|---------|--------|
| F1 | Пустые универсальные цитаты | Подходят любому в любой день |
| F2 | «Вода» без образа | Не создаёт сцену в голове |
| F3 | Абстракции без жизненного контекста | «старый способ больше не работает» — непонятно о чём |
| F4 | Шаблоны генератора цитат | «Доверяйте интуиции», «Отпустите контроль» |
| F5 | Повтор одних и тех же паттернов в одном ответе / за неделю | Ощущение одного и того же дня |
| F6 | Универсальность до бессмысленности | Можно показать **любому** человеку **в любой день года** |

**Не запрещено:** слова «Вселенная», «интуиция», «перемены» — если за ними **сцена**.

## 5. Слабый vs сильный (калибровочная таблица)

### 5.1 Одна тема — две формы

| Тема | ❌ Слабо (банально) | ✅ Сильно (есть сцена) |
|------|---------------------|-------------------------|
| Отпустить прошлое | Пришло время отпустить прошлое и открыть новую главу. | Вы можете поймать себя на мысли, что продолжаете возвращаться к ситуации, которая давно должна была остаться позади. Сегодня хороший момент перестать искать в ней новые ответы и освободить место для следующего шага. |
| Вселенная и перемены | Сегодня Вселенная будет подталкивать вас к переменам. | Сегодня Вселенная может несколько раз намекнуть вам, что пора выходить из привычного сценария — через неожиданный разговор, новую идею или предложение, которое вы обычно бы проигнорировали. |
| Интуиция | Доверяйте своей интуиции. | Если сегодня логика говорит одно, а внутреннее ощущение продолжает тянуть в другую сторону — не спешите игнорировать этот сигнал. |
| Контроль | Отпустите контроль. | Чем сильнее вы пытаетесь удержать ситуацию под контролем, тем больше она может сопротивляться. Иногда самый быстрый путь вперёд начинается с того, что вы перестаёте давить. |
| Ответ на вопрос | Вселенная подскажет правильное направление. | Сегодня вы можете получить ответ на вопрос, который давно откладывали. Не обязательно в виде озарения — иногда Вселенная говорит через случайную фразу, сообщение или разговор, который сначала покажется незначительным. |
| Перемены | Вас ждут перемены. | То, что ещё месяц назад казалось окончательным решением, сегодня может начать выглядеть иначе. Не удивляйтесь, если появится желание пересмотреть планы, договорённости или направление, в котором вы двигались последнее время. |

### 5.2 Абстракция без сцены

| ❌ | Почему слабо |
|----|--------------|
| Сегодня может появиться ощущение, что старый способ решать вопрос больше не работает. | Нет образа: какой вопрос? какой способ? что человек делает? |
| Вам нужно работать над собой. | Нет сцены |
| Ожидается трансформация внутреннего состояния. | Нет сцены |

### 5.3 Сцена есть — допустимо

| ✅ | Что рисуется |
|----|--------------|
| Сегодня будет проще заметить, какие обещания вы постоянно даёте себе и откладываете уже не первую неделю. | Внутренний список «обещаний себе», прокрастинация |
| Кто-то может снова поднять тему, которую вы считали давно закрытой. | Разговор, незакрытый конфликт |
| Вы можете неожиданно поймать себя на том, что устали объяснять очевидные вещи одному и тому же человеку. | Конкретное social micro-scene |

### 5.4 Оценка по шкале 1–10 (калибровка TL-0)

| Score | Смысл |
|-------|--------|
| 1–3 | Не показывать |
| 4–6 | Слабо, нужна правка |
| 7–8 | Хорошо |
| 9–10 | Эталон |

Примеры с разметкой — в [TODAY_LANGUAGE_CALIBRATION_V0.json](./datasets/TODAY_LANGUAGE_CALIBRATION_V0.json).

## 6. Маркеры сцены (эвристики; не замена RULE_001–003)

Не исчерпывающий список — **эвристики**, не замена трём правилам:

| Маркер | Примеры |
|--------|---------|
| **Внутренний диалог** | «поймать себя на мысли», «логика говорит одно, а…» |
| **Повторяющееся действие** | «снова возвращаетесь», «уже не первую неделю» |
| **Социальная сцена** | разговор, сообщение, «кто-то снова поднимет тему» |
| **Временная привязка** | «ещё месяц назад», «сегодня», «до обеда» |
| **Конкретный объект решения** | планы, договорённости, одно обещание себе |
| **Телесное / бытовое** | усталость объяснять, давить, откладывать |

**Anti-markers (часто = F1–F4):**

- одно короткое предложение-имperative без контекста;
- «доверьтесь / отпустите / не бойтесь» без *чего* и *где*;
- пара абстрактных существительных («трансформация», «внутреннее состояние», «ресурс») без глагола в жизни.

## 8. Порядок внедрения (калибровка → код)

**Сейчас: TL-0C.** Код (TL-1) **заблокирован** до TL-0C.1–0C.2 (паттерны) + TL-0C.3 (30+30 из api_live).

```
TL-0A/0B  corpus + auto-tags (323 logs → 2080 phrases)
  ↓
TL-0C.1  анти-паттерны (10–20 типов провалов)
  ↓
TL-0C.2  сильные паттерны + why-it-works (+ гипотеза RULE_004)
  ↓
TL-0C.3  calibration dataset (30+30 api_live, score + rules)
  ↓
TL-1     today_language_quality_v1.py
```

### Архитектура слоя (после TL-0)

```
LLM / assembler → draft text → TODAY_LANGUAGE gate (RULE_001+002+003) → show | rewrite | fallback
                                      ↑
                         calibration dataset + scene taxonomy
                         + (later) user reaction → quality memory
```

| Слой | Сейчас | Цель |
|------|--------|------|
| **TL-0 dataset** | **IN_PROGRESS** | TL-0A ✅ · TL-0B ✅ · TL-0C.1–0C.2 pending · [TODAY_LANGUAGE_CALIBRATION_V0.md](./TODAY_LANGUAGE_CALIBRATION_V0.md) · [TODAY_LANGUAGE_PATTERNS_V0.json](./datasets/TODAY_LANGUAGE_PATTERNS_V0.json) |
| Prompt hints | анти-абстракция в `today_narrative.py` | Согласовать с каноном; не единственная защита |
| RU substring ban | `_RU_NARRATIVE_BANNED_SUBSTRINGS` | Явный мусор; **не** основной фильтр |
| Contract gate | `today_contract_text_quality_v1.py` | Навигация, dedupe — **дополнение** |
| **TL-1 language gate** | **BLOCKED** | Heuristics **выведены из TL-0**, не из догадок |
| Human review | ad hoc | Продолжение TL-0 |

**При провале gate (после TL-1):**

1. retry с addon по провалившемуся RULE;
2. grounded fallback из pool **score 8+** из calibration;
3. log: `language_gate_reject`, `rules_failed`, `surface`, `generation_id`.

**Запрещено:** показывать слабый текст «как есть»; **запрещено:** писать TL-1 до calibration milestone.
