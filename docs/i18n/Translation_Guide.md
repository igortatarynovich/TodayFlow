## Варианты перевода

### Вариант 1: Использовать Google Translate в браузере (рекомендуется)

1. Откройте файл `CONTENT/i18n/en.json`
2. Найдите пустые ключи (они уже есть в `ru.json` с пустыми значениями)
3. Используйте Google Translate (translate.google.com) для перевода текстов
4. Замените пустые строки в `ru.json` на переводы

**Преимущества**: Бесплатно, быстро, хорошее качество
**Недостатки**: Требует ручной работы

### Вариант 2: Использовать DeepL API (автоматический)

1. Получите API ключ на https://www.deepl.com/pro-api
2. Установите библиотеку: `pip3 install deepl`
3. Запустите скрипт:
   ```bash
   export DEEPL_API_KEY="your-api-key"
   python3 scripts/translate_ru_batch.py
   ```

**Преимущества**: Автоматический, высокое качество
**Недостатки**: Требует API ключ (платно для больших объёмов)

### Вариант 3: Использовать Google Translate API (автоматический)

1. Получите API ключ Google Cloud Translate
2. Установите библиотеку: `pip3 install google-cloud-translate`
3. Запустите скрипт с настройками

**Преимущества**: Автоматический
**Недостатки**: Требует API ключ и настройку

### Вариант 4: Массовый перевод через CSV

1. Используйте скрипт для экспорта в CSV:
   ```bash
   python3 scripts/export_for_translation.py
   ```
2. Откройте CSV в Excel/Google Sheets
3. Используйте формулу Google Translate или встроенный переводчик
4. Импортируйте обратно:
   ```bash
   python3 scripts/import_translations.py translation_batch.csv
   ```

## Проверка качества

После перевода запустите валидацию:
```bash
python3 scripts/validate_i18n.py
```

Убедитесь, что:
- Нет пустых значений
- Сохранена терминология (паттерны, оси, фазы)
- Тон соответствует проекту (luxury, осознанность)
- Нет прямых переводов с английского
