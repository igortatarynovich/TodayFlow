# TodayFlow Lexicon (Web Canon v1)

Курируемый смысл с метаданными. Не список фраз — каждая фраза имеет `context`, `trigger`, `emotion`, `reaction`, `tone`.

**lexicon.json** содержит:
- **banned_words** — стоп-слова (порталы, вибрации, энергии…). Quality Gate отклоняет контент при их появлении.
- **tags_allow_list** — теги только из этого списка.
- **phrases** — фразы-наблюдения с метаданными для комбинации/адаптации.

Мы не генерируем смысл — мы его курируем.
