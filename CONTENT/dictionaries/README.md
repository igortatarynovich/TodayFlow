# Словари маркеров (Web Canon v1)

Типизированные элементы для блока «живой» текст в DailyForecast. В прогнозе используются **только** ключи из этих словарей.

- **body_markers.json** — телесные маркеры (сон, голова, плечи, аппетит, напряжение…)
- **social_markers.json** — социальные (сообщение, тон, игнор, дедлайн…)
- **domestic_details.json** — бытовые детали (дорога, покупки, кухня, список дел…)
- **micro_actions.json** — микро-действия (пауза 30 сек, вода, одно письмо…)

Quality Gate проверяет, что каждый маркер в прогнозе присутствует в соответствующем словаре.
