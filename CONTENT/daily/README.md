# Daily Templates

Daily templates provide structured content for daily insights, practices, prompts, and guidance.

## Structure

Each daily template contains:

- `template_id` - Unique identifier (e.g., `DAILY.AFFIRMATION.001`)
- `type` - Template type: `AFFIRMATION`, `DIGEST`, `FOCUS`, `PATTERN_TODAY`, `PRACTICE`, `PROMPT`, `WARNING`, `WIN`
- `category` - Category for grouping
- `text` - Original/semantic text
- `human_text` - Human-readable version (from Human Layer)

## Types

### AFFIRMATION (150 templates)
Positive affirmations for daily support.

### DIGEST (80 templates)
Daily digest/summary texts.

### FOCUS (80 templates)
Daily focus prompts and guidance.

### PATTERN_TODAY (200 templates)
Daily patterns for EP, RL, CR, MS layers.

### PRACTICE (208 templates)
Daily practices (breathing, meditation, journaling, grounding, body_scan, movement, etc.).

### PROMPT (150 templates)
Reflection and journaling prompts.

### WARNING (60 templates)
Daily warnings and cautions.

### WIN (60 templates)
Daily win/celebration templates.

## Human Layer

The `human_text` field is generated by the Human Layer system, transforming semantic instructions into human-readable text.

Example:
- `text`: "Focus on what you can control today."
- `human_text`: "Focus on what you can control today. Let go of what's outside your influence, and direct your energy toward actions that matter."
