# PWA Icons

Для работы PWA необходимо создать иконки следующих размеров:

- `icon-72x72.png` - 72x72px
- `icon-96x96.png` - 96x96px
- `icon-128x128.png` - 128x128px
- `icon-144x144.png` - 144x144px
- `icon-152x152.png` - 152x152px
- `icon-192x192.png` - 192x192px (обязательно)
- `icon-384x384.png` - 384x384px
- `icon-512x512.png` - 512x512px (обязательно)

## Требования к иконкам:

1. **Формат:** PNG
2. **Прозрачность:** Поддерживается
3. **Maskable:** Иконки должны быть maskable (безопасная зона для обрезки)
4. **Стиль:** Соответствовать бренду TodayFlow

## Генерация иконок:

Можно использовать онлайн-генераторы:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://maskable.app/

Или создать вручную из исходного логотипа TodayFlow.

## Временное решение:

До создания финальных иконок можно использовать placeholder изображения или логотип из `/images/logos/`.
