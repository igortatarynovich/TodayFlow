## Guidelines

- **Subtlety**: Banners should enhance, not dominate
- **Text Overlay Ready**: Leave space for text or use semi-transparent areas
- **Responsive**: Provide multiple sizes or use responsive images
- **Brand Consistency**: Match slowlife aesthetic
- **Note**: Currently using CSS-based CTAs, banners are optional enhancements
