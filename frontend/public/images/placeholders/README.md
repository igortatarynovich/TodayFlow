# Placeholder Images

## Loading Placeholders

### Card Placeholder
- **File**: `placeholder-card.webp`
- **Size**: 400x300px (matches practice/pattern cards)
- **Usage**: Skeleton loaders, loading states
- **Format**: WebP
- **Style**: 
  - Blurred, low-opacity version of actual cards
  - Or simple gradient/color placeholder
- **Note**: Can use CSS blur on actual images instead

### Avatar Placeholder
- **File**: `placeholder-avatar.webp`
- **Size**: 64x64px
- **Usage**: Avatar loading state
- **Format**: WebP
- **Note**: Code generates initials, placeholder optional

## Empty State Illustrations

- **Style**: Simple, friendly, minimal illustration

### Generic Empty
- **File**: `empty-generic.svg`

## Guidelines

- **Simplicity**: Placeholders should be simple and clean
- **Brand Consistency**: Match slowlife aesthetic
- **Size**: Match target image dimensions
- **Format**: SVG for illustrations, WebP for blurred placeholders
- **Usage**: Provide visual feedback during loading or empty states
