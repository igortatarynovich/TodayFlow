# Decorative Patterns

## Grain/Texture Patterns

### Paper Grain
- **File**: `pattern-grain.svg`
- **Size**: Tileable SVG (100x100px repeat)
- **Usage**: Subtle texture overlay (currently `body::before` CSS)
- **Opacity**: 5-10% when applied
- **Style**: Paper-like texture
- **Format**: SVG

### Noise Pattern (Alternative)
- **File**: `pattern-noise.svg`
- **Size**: Tileable SVG
- **Usage**: Alternative texture option
- **Format**: SVG

## Border Patterns

### Frame Border
- **File**: `border-frame.svg`
- **Size**: Scalable SVG
- **Usage**: Card borders, decorative frames
- **Format**: SVG
- **Style**: Minimal, elegant

### Divider Pattern
- **File**: `border-divider.svg`
- **Size**: Scalable SVG (horizontal repeat)
- **Usage**: Section dividers
- **Format**: SVG

## Guidelines

- **Subtlety**: Patterns should be very subtle
- **Tileability**: Must tile seamlessly
- **Performance**: SVG is preferred for scalability
- **Usage**: Patterns are decorative, should not distract
- **Current**: Most patterns implemented via CSS, SVG patterns are optional
